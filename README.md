# Rama

Roma convert movie to a sprite image.

## Requirements

- FFmpeg
- Imagemagick

## Installation

Add this line to your application's Gemfile:

    gem 'rama'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rama

## Usage

```
rama sprite -i ~/movies/test.mp4 -o ~/images/result.jpg
```
